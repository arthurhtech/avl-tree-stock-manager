# AVL TREE Stock Manager

Um sistema de gerenciamento de estoque via linha de comando desenvolvido em Python. O objetivo do projeto foi demonstrar a implementação de uma **Árvore AVL ** para armazenar e manipular dados de produtos.


## Funcionalidades

1.  **Adicionar Item:** Insere produto com ID, Nome, Quantidade e Preço. Verifica duplicidade de ID e rebalancea a árvore automaticamente.
2.  **Buscar Item:** Localiza um produto pelo ID instantaneamente usando busca binária.
3.  **Relatório Ordenado:** Exibe todos os produtos ordenados por ID
